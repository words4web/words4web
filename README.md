# Words4Web Website

A modern React web application built with Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or pnpm

### Installation

1. Install the dependencies:

   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To build the application for production:

```bash
npm run build
```
